GAE user management with gaeusers

The gaeusers module offers you a basic user management for apps which running on App Engine Cloud.

from gaeusers import *

options = {'appid': 'gaeusers',
           'mailstring': 'your_name <your_email>',
           'crypt':'md5',
           'crypt_rounds': 10,
           'password_salt':'xyz987'}

gaeusers = GaeUsers(options)
Parameter Description
appid Application id
mailstring String with name and email
crypt md5 | sha1
crypt_rounds How often the password gets crypt.
password_salt Salt with gets added to the password.

After creating an instance of gaeusers, it offers the following methods.

Register 
response = gaeusers.register(email)

Responses:

dict(register=False, error="not equal")
dict(register=False, error="not valid")
dict(register=False, error="present")
dict(register=True, key="<userkey>")
dict(register=False, error="empty")
Login 
response = gaeusers.login(email, password)

Responses:

dict(login=True, key="<userkey>")
dict(login=False)
dict(login="not active")
dict(login="unknown")
dict(login="empty")
Conform 
response = gaeusers.conform(confirm_link)

Responses:

dict(response=True)
dict(response=False)
Lose password 
response = gaeusers.lose_password(email)

Responses:

dict(response="empty")
dict(response=True)
Change password 
response = gaeusers.change_password(key, password_old, password_new, password_renew)

Responses:

dict(response="change")
dict(response="not equal")
dict(response="wrong")
Users model 
class Users(db.Model):
    email = db.StringProperty(required=True)
    password = db.StringProperty()
    active = db.BooleanProperty()
    created = db.DateTimeProperty(auto_now_add=True)
    logins = db.IntegerProperty()
    lastlogin = db.DateTimeProperty(auto_now=True)
ActiveLinks and PasswordLinks models 

class ActiveLinks(db.Model):
    user = db.StringProperty()
    link = db.StringProperty()

class PasswordLinks(db.Model):
    user = db.StringProperty()
    link = db.StringProperty()
