GAE user management with gaeusers

The gaeusers module offers you a basic user management for apps which running on App Engine Cloud.

from gaeusers import *
options = {'appid': '', 'mailstring': 'your_name <your_mail>', 'crypt':'md5'}
gaeusers = GaeUsers(options)
Parameter Description
appid Application id
mailstring String with name and email
crypt md5 | sha1

After creating an instance of gaeusers, it offers the following methods.

register_response = gaeusers.register(email)
Responses: 
{"response":{"check":"not valid"}}
{"register":{"check":"present"}}
{"register":{"check":"true", "key": ""}}
{"register":{"check":"empty"}}
login_response = gaeusers.login(email, password)
Responses: 
{"login":{"check": "true","key": ""}}
{"login":{"check": "false"}}
{"login":{"check": "not active"}}
{"login":{"check": "unknown"}}
{"login":{"check":"empty"}}
confirm_response = gaeusers.conform(confirm_link)
Responses: 
{"conform": "true"}
{"conform": "false"}
losepassword_response = gaeusers.lose_password(email)
Responses: 
{"response":{"send":"true"}}
changepassword_response = gaeusers.change_password(key, password_old, password_new, password_renew)
Responses: 
{"response":{"check":"change"}}
{"response":{"check":"not equal"}}
{"response":{"check":"wrong"}}

How to use it, is shown in main.py templates using the bootstrap framework.
